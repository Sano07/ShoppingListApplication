Shopping List App

Приложение для управления списками покупок.

**Stack:** Kotlin, Jetpack Compose, MVVM, Room, Hilt, Coroutines, Flow, Navigation Compose  

Архитектура:
UI → ViewModel → Repository → DAO → Room  

- Работа с локальной БД (Room)  
- Реактивные обновления через Flow  
- DI через Hilt  
- Обработка UI-событий через sealed class
